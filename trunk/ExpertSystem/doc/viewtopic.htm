Você provavelmente já percebeu que as IDE’s Java normalmente conseguem, a partir de uma classe, exibir seus métodos, atributos, modificadores etc. Pois bem, onde está o segredo desta tarefa? A resposta é : Reflection API.

A Reflection API reflete as classes, interfaces, e objetos na Java Virtual Machine corrente, esta API pode ser utilizada em ferramentas para debug, navegação em classes e etc. Com a Reflection API podemos determinar a classe de um objeto, obter informações sobre a classe, invocar um método dentre outras tarefas.

Neste tutorial mostraremos uma dica para recuperar os modificadores e métodos de uma classe com os seus respectivos parâmetros e tipos de retorno.

O pacote que contém as classes necessárias para o Reflection é java.lang.reflection. Usaremos a seguinte classe sendo a classe que terá os seus métodos exibidos em nosso exemplo:



Código:
public final class ClasseQualquer {
  public String umMetodo(String s){
     return "Voce informou a String : " + s;
   }

public String outroMetodo(String s){
  return "Voce informou a String : " + s ;
   }

public int outroMetodo(int i, String s){
  return i ;
   }
}




O nosso programa, que mostrará as informações de uma determinada classe, é o TesteReflection. O interessante é que as classes a serem exibidas são passadas para os métodos como String.

Em nossa dica utilizamos constantemente a classe Class do pacote java.lang, que representa interfaces e classes em uma aplicação java. A Class possuí métodos que nos permite, por exemplo, carregar uma classe a partir do nome, criar uma nova instância de uma classe, obter os métodos, dentre outros.

O trecho de código responsável por carregar a classe a partir de uma String é o seguinte:



Código:
Object object = null;

try {
  Class classDef = Class.forName(nomeClasse);
     //Carrega classe a partir de um nome
  object = classDef.newInstance();
     //Instancia e atribui a instância ao objeto object.

} catch (InstantiationException e) {
  System.out.println(e);

} catch (IllegalAccessException e) {
  System.out.println(e);

} catch (ClassNotFoundException e) {
  System.out.println(e);
}




Para obtermos os modificadores da classe fazemos uso do seguinte trecho de código:


Código:
/*
cl foi declarado na classe como sendo do tipo Class,
a partir do object utilizamos o método getClass para fazermos
referencia a classe especificada como parâmetro para o método.
*/

cl = object.getClass();

/*
getModifiers retorna os modificadores de acesso codificados em um inteiro.
*/
int mod = cl.getModifiers();

String sMod = "";

/*
Modifier é uma classe do Pacote java.lang.Reflect, o mesmo possuí
alguns métodos que recebem como parâmetro um inteiro codificado
que é retornado pelo método getModifiers.
*/

if (Modifier.isPublic(mod))
  sMod += "public ";

if (Modifier.isAbstract(mod))
  sMod += "abstract ";

if (Modifier.isFinal(mod))
  sMod += "final ";

System.out.println("O Modificador da Classe " +
                          """ + cl.getName() + """ +
                          " é : " + sMod + " ");




Para obtermos os métodos de uma classe fazemos o uso do seguinte trecho de código:


Código:
/*
Method é uma classe da Reflections API que contém
as informações sobre os métodos de uma classe.
*/

Method[] metodos = object.getClass().getDeclaredMethods();

/*
getDeclaredMethods retorna um array de métodos declarados
pela classe, os métodos herdaddo não são retornados, a partir
daí basta percorrer todo a array e listar os métodos.
*/

System.out.println("Os Métodos da Classe São : ");
for (int i = 0 ; i < metodos.length ; i++){
  System.out.println("Nome : " + metodos[i].getName());
     // Para cada método nós podemos recuperar os parâmetros e os tipos.
  Class[] param = metodos[i].getParameterTypes();

//getParameterTypes retorna um array com os tipos dos parâmetros.
for (int p = 0; p < param.length ; p++){
  System.out.println(" Parametro "+ p +" : "+param[p].getName());
}

System.out.println(" Tipo de Retorno : "+
                          metodos[i].getReturnType().getName());

//getReturnType retorna o tipo de retorno do método.
System.out.println("---------------------");

}
